# Special Tic-Tac-Toe

This is a variation of the traditional Tic-Tac-Toe game where a draw is not possible. The game rules are slightly modified to ensure that one player will always win.

## Features

- No possibility of a draw
- Simple and intuitive interface
- Implemented using HTML, CSS, and JavaScript

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/ImComy/special-tic-tac-toe.git
    ```
2. Navigate to the project directory:
    ```bash
    cd special-tic-tac-toe
    ```
3. Open `index.html` in your browser to start the game.

## Usage

- Player 1 uses 'X' and Player 2 uses 'O'.
- only 3X's / 3O's are avaliable and the very first X/O will change its place to a place you decide
- Take turns clicking on the grid to place your mark.
- The game will automatically declare a winner when a player gets three marks in a row, column, or diagonal.
- Enjoy the game without the possibility of a draw!
